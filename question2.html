objects in Javascript

objects:-

Objects can be considered as the Unordered collection of data stored in the form of Key : Value Pairs.These keys can be functions or variables.

variables can also contain many values.Objects are variables too. But objects can contain many values.

Eg:- let person = {firstName:"Leo", lastName:"Dass", age:50, eyeColor:"Brown"};

objects in Javascript:-

Basically,JavaScript is an object-based language. Everything is an object in JavaScript.A JavaScript object is a collection of named values.And it is a template based not class based. Here, we don't create class to get the object. But, we direct create objects.

Creating objects in Javascript

There are 3 ways to create object in JS,

1) JavaScript Object by object literal

This is the easiest way to create a JavaScript Object.Using an object literal, you both define and create an object in one statement.And the syntax is,

Eg:-object={property1:value1,property2:value2...propertyN:valueN}


2) By creating instance of Object

Here, new keyword is used to create object.And the syntax is,

var objectname=new Object();


3) By using an Object constructor

Here, we need to create function with arguments. Each argument value can be assigned in the current object by using this keyword.The this keyword refers to the current object.

 Eg:-
function emp(id,name,salary){  
this.id=id;  
this.name=name;  
this.salary=salary;  
}  
e=new emp(103,"vijay",30000); 
console.log(e);
